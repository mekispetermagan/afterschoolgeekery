Blooming flower
What is this project about?

In this project, kids draw a flower, and add code that makes it bloom. It starts from very small, and grows to normal size and beyond:

You can see the end result here:

Step 1: Drawing

The trick is that we draw the green parts (the stem and leaves) and the colorful flower part (the blossom) separately as different sprites. Use the circle tool and the line tool:

Make sure the flower part is well centered, so that it won't change its position when growing:

When you are ready, place the two sprtes in the stage so that they align perfectly. Make sure the flower part is the last you move, so that it will be in the front:

Step 2: Coding

The idea is that the flower part starts as very small, and then grows at each mouse click. This is a good exercise at when to use a "set" block and when its "change" counterpart. At the beginning, the flower is set to a small size:

The flower changes its size at each mouse click. You can make the growth simple, or use a "repeat" block for a smoother growth:

...And you are ready! :) Look how simple and amazing your animation is.
