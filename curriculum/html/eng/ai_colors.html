Color Mixer
What is this project about?

This project demonstrates how to mix any color out of three basic components: red, green, and blue. Colors mixed this way are called RGB colors, after the first letters of the names of the three basic solors. Colors are mixed with three sliders, and displayed in a label.

Since both the interface and the code are rather simple, a significant part of the project work is about discussing how color mixing works, and experimenting with it.

Step 1: Discussion and experimenting

Start asking the participants: What are RGB colors? What does RGB stand for? Whether or not kids have any idea about this, open one of the various web applications available that let users mix RGB colors with three sliders. (Although other methods, like color wheels, are also useful, please stick to RGB sliders, since the rest of the project work will use this method.)

Since websites come and go, we provide three alternatives. In the unlikely case that none of these works by the time you teach this project, search the internet for alternatives (and let us know that we should refresh this guide).
