What is it about?

This is a very simple project with random costume change.

Here's the end result of the project work:

Step 1: Preparing the box

Say goodbye to Scratchie, and choose the "Gift" sprite, which is a gift box.

Now switch to the "Costumes" tab. The box comes in two colors: dark red and blue. Keep the one that you like more; or use the color fill tool to repaint it. Now that you have a single costume, create a duplicate before you alter anything, because we will need the original, too:

Now let's open the box on the duplicate costume. Select and deelte the bow at the top. Then draw a dark rectangle, and shape it to fit the top of the box. Finally, create a few duplicates of the costume with the open box.

Step 2: Filling the box

Now select a new costume. Not a new sprite, just a new costume. Choose whatever you want to see in the box. Copy and paste it to one of the open box duplicates, and shrink it so that it will fit in the box.

If you want a perfect fit, use the eraser tool to shape the content before you place it in the box.

Repeat this as many times as you like! :)
