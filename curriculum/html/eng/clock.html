What is this project about?

In this project kids will use Scratch code to make a pre-designed analog clock work. See the final result here:

With kids who already have some grasp of the ideas of angles and degrees, the project work can be completed in 30 minutes. In 60 minutes you will have enough time to introduce and practice these concepts.

Similarly, if kids in your group are not hundred percent confident about reading time from an analog clock, this project is a good opportunity to practice this skill. However, if both clocks and degrees are completely alien to them, the project work may take longer, and you may risk overwhelming them.

Preparations

Present the group the final version of the project. Then let everyone open and remix this initial version. It contains all the sprites and graphics you need for the project work:

Before starting the project work, discuss the functions and the different hands, and for a few minutes practice setting up different times. Kids take turns telling different times of the day, and you set the clocks together. Meanwhile, you can clarify that all you need for this project is setting the "direction" property of the three hand sprites. Click into the value of the direction tool under the stage, and use the pointer to turn the hands of the clock in the required directions. Meanwhile, you can read the corresponding degrees: for example, 0 degrees is 12 o'clock in case of the hours hand; and 90 degrees is 15 second in case of the seconds hand.

Finally, if it feels necessary, you can present to the kids this project demonstrating how degrees represent directions, from from 0 to 360. (Unfortunately, Scratch uses negative degrees for directions towards the left, which can be confusing for smaller kids.)

Step 1: Setting the seconds

It is a good idea to start coding with the seconds hand, because this is moving visibly, so that the first step will already provide kids with a feeling of success. First you need to figure out how you can get the current time from Scratch. The block reporting the current hour, minute, and second is a bit hidden, but with the appropriate questions you can help the kids finding it. Where are the blocks related to time? Except for the "wait" block which is in the Control group, all of them are in Sensing. Can you remember that some blocks open up when clicked, and shows options? Which block will show the "hours, "minute", and "second" options? Found it? Use it!

Now that you have the key block, the rest is just simple math. You have already concluded that coding a clock is all about directions. A full circle is 360 degrees, and it is 60 seconds. Thus, one second is 360/60 degrees, that is, 6 degrees. The direction of the hand is 6 times the number of seconds passed in the current minute:

If mental calculation is a problem, feel free to use the division block (a computer's job is to compute after all):
