Getting started with App Inventor
Logging into the web app

Creating a project

To create your first project, click on the "New project" button:

Give your project a name, and use the default values for "Toolkit" and "Theme". Note that you can't use accented or cyrillic letters, or extra symbols in the project name. Only letters of the English alphabet, arabic numbers, and underscores are allowed:

The Designer interface

Once your new project is created, you'll find yourself in the Designer interface. It has several parts:

  1. The old stylized smartphone in the middle (marked with purple here) is called the Viewer. This is where you will drag and organize your app's components.
  2. On the left you find the Palette (marked with red). This is a list of components that you can drag to the middle and use in your app.
  3. The Components area, right of the Viewer (marked with dark blue) shows all the components you've selected for your app, organized as a tree. You can also rename and delete components here. At the beginning, you have a single base component, "Screen1", the opening screen of your app.
  4. The Properties area on the right (marked with light blue) is where you can stylize a component that you selected in the Components area. You can set background color, width, height, and various other properties, depending on your component.
  5. There are two important buttons at the top: "Connect" and "Build". These will make your project actually work on a mobile device. We will discuss them later.

To begin with, drag a button from the Palette to the Viewer. It will appear at the top left corner of the stylized mobile:

Use the Properties interface to set a few properties of the button. For example, set the background color to dark green clicking on "Custom color", the font size to 30, the width to 50%, the height to 30%, the shape to rounded, the text to "Click here!", and the text color to white.

The Blocks interface

It's time to add some behavior to your button with code. Just like we build code from blocks to animate sprites in Scratch, we builld code from blocks to add behavior to components in App inventor. You can switch to the Blocks interface at the top right:

The Blocks interface has two main parts:

  1. The Blocks area on the left (marked with red) contains several drawers: built-in drawers for general blocks, and component-specific drawers for the components' blocks.
  2. The Viewer area on the right (marked with green) is where you can snap blocks together to build your code.

Just as in Scratch, the code always starts with an event block. Each component has its events. The main event of a button component is "click"; that is, the event that the button is pressed.

For simplicity, let's change the background color of the button on click. For this, you will need to add a "set" block from the button's drawer, and a color from the built-in "Colors" drawer:

Connect to your mobile device

It's time to test your app on a mobile device. To do this, you will need the App Inventor Companion. You can find it in the Play Store if you use Android, or in the App Store if you use Apple:

Once the app is installed, you are ready to scan the qr code from your computer screen:

To get the qr code itself, click on the "Connect" button at the top of your computer screen, and select "AI Companion":

Once you've scanned the code, the app will start on your mobile device. Check if whether button really changes its color when you tap on it:

Admittedly this is not the most high tech mobile app, but it works! :)

Install on your mobile device

To actually install the app on an Android device, you will need to scan another qr code. First, reset the Companion app from your computer, under "Connect". Then use "Build" to create the qr code to install the app:

After scanning the qr code, your device will give you several warnings. These are absolutely justified; you should be extra careful installing an app downloaded from an untrusted location. However, this app was developed by you, and you didn't put anything harmful in it; neither did App Inventor, a web application maintained by the same institute as Scratch, and trusted by millions of users. Feel free to assure your device that th app is safe to install. After installlation, the app's icon will appear on your mobile screen, and you can run it whenever you want:

Note: currently App inventor apps can only be live tested, but not installed on Apple devices. In the unlikely case that some of your participants have access to an iPhone, but not to an Android device, we recommend to use Thunkable, a cross-platform alternative.
