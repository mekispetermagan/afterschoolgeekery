What is Scratch and what is it for?
About Scratch


Scratch is a programming language, a tool to explain to a computer what it should do. Scratch's player (interpreter) obediently executes your instructions. So you can program animations and video games, tell a story, or even make a presentation; Scratch makes it quick and easy to put together spectacular projects.

Scratch is a visual programming language in which instructions to the machine are not typed in a text document, but put together on the screen from color blocks. The color and shape of the blocks reflect their functions, so it's very easy to learn what they are for.

Scratch has been developed by the world's leading university of engineering, the Massachusetts Institute of Technology in the US, for over 20 years, and is used by hundreds of millions of people worldwide to teach programming to 8 to 16 year olds (a version for younger kids called Scratch Junior is also being developed).

Why is Scratch good?
  • Because Scratch turns programming into a game. Kids don't even realize they're learning.
  • Because the visual interface makes programming easily accessible for those with literacy difficulties.
  • Because the emphasis is not on the math and technology side of programming, but on creativity. We can give children who have had only failures in math or IT education at school a positive learning experience.
  • Because, all in all, Scratch is a great tool for software developers and a great way to prepare children to learn modern programming languages.
How do you get started?
What will you need?


The easiest way to access Scratch is online on the Scratch website but if you don't have a stable internet connection you you can also download the Scratch app. In this guide, we'll cover the online interface, but the downloaded and installed software works exactly the same way.

First of all, you will need laptops; one for each child, and one for you, the teacher. If you don't have a laptop, a desktop computer is also fine. You can use Scratch on a tablet with a large enough screen (at least 8 inches), but in our experience working on a smaller tablet or phone is inconvenient.

It's a good idea to have a mouse for the laptops; children's hands are often not yet dexterous enough to use the touchpad effectively. You'll also need a projector or a large monitor that the kids can watch while you're working.

The Scratch interface

After registering, you should go straight into Scratch and create your first project, which will already be saved in the system.You can start a new project by clicking on "Create".

Scratch's interface is a bit complex at first glance, but it's very intuitive and easy to navigate:

1. On the header marked in yellow you can

  • click on the orange button to share a project
  • click on the folder icon to switch to a previous project
  • download or upload younger project
  • change language
and you can find many other useful things here, feel free to explore the possibilities!

2. n the section marked orange, you will find Scratch blocks of different colors. You can use them to build the program code. In the following tutorials you will gradually get to know them.

3. The large empty area marked in green is called script area; you can build code from the blocks here. Blocks placed here with the mouse will automatically connect if they are close enough together.

4. The part marked in blue is the Scratch stage. This is where the program you are creating will run, filling the stage with spectacular animations. You can most often start a program by clicking on the green flag icon, and stop it by clicking on the stop-sign icon. Click on the top right icons to enlarge or shrink the stage as required.

5.In the purple area below the stage, you can choose which sprite you want to work with. Scratchie kitten is the only sprite to begin with, but you will work with more than one sprite in almost every project. You can add a new sprite to a project by clicking on the kitten head icon at the bottom right, and you can add a new background by clicking on the small picture icon.

In Scratch, we use program code to give tasks to sprites (and possibly the background). Before editing a sprite's code, click on its thumbnail to select it. In the same area, you can set the basic properties of the sprites under the stage. Feel free to try them out!

Finally, there are two more important tools to explore right from the start. The first is the graphic editor, which you can open by clicking on the "Costumes" tab in the block interface at the top left. This offers a wealth of options, which we will explore in a separate guide.

The other is the sound editor, which opens by clicking on the "Sounds" tab in the same window. This has much more limited options, but well-chosen sounds can really enhance the experience of a project.
