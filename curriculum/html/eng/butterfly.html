Butterfly
What is this project about?

This is a very simple little animation that starts from a single drawing, but it's quite spectacular. You can see the end result here:

Step 1: Drawing the butterfly

Click on the little bin icon to delete Scratchie the cat. Hover the mouse over the kitten head icon in the bottom right corner and click on the small brush in the menu that appears to draw a new sprite.

If you don't feel like drawing, you can skip this whole section by clicking on the magnifying glass icon instead of the brush, and selecting the finished butterfly sprite.

When the graphic editor appears, you can start drawing as usual, using the circle tool. It is a good idea to start with the butterfly's body and head. When you are more experienced, you can color them with a gradient.

Now we're ready for one wing. You can also color this with a gradient. Use the reshape tool to get the final shape, and draw nice colorful patterns on it.

Once you're happy with one wing, you can move on to the other. This one is ready in one step, as you just need to copy and the one that is ready. Select the wing, then click on the copy and paste icons above. Drag the second wing into place, and flip it horizontally using the flip icon at the top.

If you like, you can even perfect the result by using the line tool and the reshaping tool to add tentacles on it.

Step 2: Phases of the movement

Duplicate the costume, then mark and narrow the wings of the butterfly a little. Repeat this a few times until they are completely thinned. This is how the butterfly closes its wings.

And if you do the same thing in reverse, the wings will open. By the end of the process you will have ten to twelve costumes.

Step 3: The program code

After all that drawing, it will be nice to do some coding. The first step is to make the butterfly smaller, so that it has room to fly around the stage.

Now for the wing movement code. This will be a simple costume change with a bit of delay.

The code for flying is just as simple: you glide the sprite to a random position over and over again.

Finally, choose a nice background behind the butterfly. And don't forget to give each other a demonstration! :)
